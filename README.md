# CarsNFS
## Mobile app for Android
![Screen](https://www.dropbox.com/s/ofv5yogfh19wqqe/1.png?raw=1)
![Screen](https://www.dropbox.com/s/c16afmi0hskk4v5/2.png?raw=1)


Cars NFS 'то репозиторий, содержащий приложение, которое отображает список автомобилей из игры NFS в виде списка при помощи RecyclerView и вложенного RecyclerView отображающего марки автомобиля горизонтальным списком. Приложение позволяет прокручивать список автомобилей, нажимая на горизонтальный RecyclerView, чтобы перейти к нужному автомобилю. 
При нажатии на автомобиль открывается страница с его фотографией и информацией.

## Функциональность
* Отображение списка 20 популярных в реальном мире автомобилей из культовой серии игр NFS.
* RecyclerView - для отображения списка автомобилей с названием и фото.
* Вложенный RecyclerView - для прокрутки марок автомобилей с возможностью при нажатии перейти к нужному автомобилю.
* Открытие страницы с фотографией и подробной информацией об автомобиле при нажатии на него в основном списке.

## Зависимости
+ RecyclerView: Используется для отображения списка всех автомобилей. Каждый элемент списка представлен в виде карточки, содержащей информацию о типе марке,
модели и изображение автомобиля. 
+ Вложенный RecyclerView: Используется для горизонтальной прокрутки списка марок представленных автомобилей. Нажатие на элемент горизонтального списка произведет скролл к нужному автомобилю в
основном списке.
